# Screenshots

Add your screenshots here to showcase the Voice Assistant in action.

## Suggested Screenshots

1. **Startup Banner** — The assistant's welcome screen in the console.
2. **Greeting Response** — The assistant responding to "Hello" or "Good Morning".
3. **Time Response** — The assistant telling the current time.
4. **Date Response** — The assistant telling today's date.
5. **Google Search** — The assistant opening Google search in the browser.
6. **Exit Response** — The assistant saying goodbye.
7. **Error Handling** — The assistant handling unrecognised speech gracefully.

## How To Take Screenshots

- **Windows:** Press `Win + Shift + S` to open Snipping Tool.
- Save the screenshots in this folder with descriptive names, e.g.:
  - `01_startup_banner.png`
  - `02_greeting_response.png`
  - `03_time_response.png`
  - `04_date_response.png`
  - `05_google_search.png`
  - `06_exit_response.png`
  - `07_error_handling.png`
